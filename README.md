# deepsleep
deepseek异父异母的亲兄弟（bushi）一个小整活

deepsleep对于任何的输入都会在思考两秒半后输出“服务器繁忙请稍后重试”

但是你可以告诉他“醒一醒”来真正激活他。

![image](https://github.com/user-attachments/assets/fee53d5e-9ecc-4a62-9de5-4a317e092949)

激活后目前使用的是zhipuai

修改时，在zhi_deepsleep.py的九十行处提供你的api
